## 1. MPOps?

- ML 주기
  - 문제정의 -> 데이터 가져오기 -> 데이터 탐색 -> 데이터 가공 -> 데이터 검증 -> 데이터 나누기 -> 학습 방법 결정 -> 모델 만들기 -> 모델 검증 -> 대규모 학습 -> 대규모 배포 준비 -> 배포/운영 -> 로깅 -> 모니터링(데이터 가져오기부터 반복)
- Who?
  - DS, DA vs DE, SE
- DevOps와의 비교
  - 학습 모델을 Training하고, 배포하는 단계가 추가로 존재.
  - 데이터을 입력 받음.
- Why?
  - Rapid Development
    - 재학습/재배포 자동화
